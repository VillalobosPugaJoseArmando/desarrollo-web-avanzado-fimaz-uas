# Práctica 1 – Parcial 2

## Objetivo
Implementar un CRUD completo de productos utilizando PHP con PDO y Programación Orientada a Objetos (POO). Se desarrolló una arquitectura en capas separando la lógica de conexión a base de datos (`Database`), el modelo de datos (`Producto`) y el controlador (`ProductoController`), con una interfaz web construida con Bootstrap 5.

## Tecnologías utilizadas
- PHP 8+
- MySQL
- PDO (PHP Data Objects)
- Bootstrap 5
- POO (Programación Orientada a Objetos)

## Instrucciones de ejecución
1. Importar la base de datos `escuela` en MySQL
2. Asegurarse de que la tabla `productos` exista con los campos: `id`, `nombre`, `descripcion`, `existencia`, `precio`
3. Ejecutar el servidor con el comando:
```
php -S localhost:8000
```
4. Abrir el navegador en `http://localhost/index.php`
