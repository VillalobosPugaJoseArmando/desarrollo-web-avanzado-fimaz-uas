# Práctica 2 (Actividad 3)– Parcial 2

## Objetivo

Demostrar el uso de transacciones en PDO con `beginTransaction()`, `commit()` y `rollBack()`.

## Tecnologías utilizadas

- PHP 8+
- MySQL
- PDO

## Instrucciones de ejecución

1. Tener XAMPP con Apache y MySQL activos.
2. Colocar el proyecto en `htdocs/practica-transacciones`.
3. Abrir: `http://localhost/practica-transacciones/`
